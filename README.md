# supplier-sales-analysis  

資料庫取得來源：臺灣大學資訊系統訓練班線上課程  
此專案使用模擬電商資料庫進行資料分析練習。

## 資料庫TABLES

-Clients  
-Employees  
-Products  
-Sales  
-salesall  

## 使用工具  
-SQL  
-Excel  
-Power BI 

## SQL分析案例  

01.產品資料驗證  
02.供應商銷售排行  
03.每月營收匯總  
04.業績達標分析  
05.訂單內產品金額排名(window function)
